
# Installation Instructions

## 1. Download
```
	git clone https://github.com/Raviyanto/Emacs_Config.git
```
## 2. Install Fonts
```
	$ cd Emacs_Config
	$ cp -R bitstream_vera_mono ~/.local/share/fonts
	$ cd $HOME
	$ fc-cache -f -v
```
## 3. Install Emacs Configuration
```
	$ cd Emacs_Config
	$ cp -R moe-theme.el vendor ~/.emacs.d
	$ cp .emacs $HOME
```
